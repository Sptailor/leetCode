# LeetCode Practice

This folder contains solutions and practice work for LeetCode problems.

## Contents

Solutions are written in Java and cover various topics including:

- **Arrays & Strings**: Two Sum, Valid Anagram, Group Anagrams, Longest Consecutive Sequence
- **Linked Lists**: Reverse List (iterative & recursive), Merge Lists, Add Lists, Detect Cycle
- **Stacks & Queues**: Valid Parentheses, Min Stack, Daily Temperatures
- **Trees**: Invert Tree, Same Tree, Max Depth, Valid BST
- **Hash Tables**: Contains Duplicate, First Unique Character, Intersection
- **Two Pointers**: Two Sum (sorted), Container With Most Water, Valid Palindrome
- **Recursion**: Factorial (iterative & recursive)
- **Greedy**: Best Time to Buy and Sell Stock, Majority Element
- **Miscellaneous**: Snake Case Converter, Rate Limiter

## Structure

Each file contains a standalone solution to a specific LeetCode problem. File names generally correspond to the problem being solved.
